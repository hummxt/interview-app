# Interview Preparation App

This mobile application is designed to help users practice for technical interviews. The platform provides a personalized experience through an onboarding process where users can select their skill levels and learning paths. It includes features for practicing with technical questions and tracking preparation progress.

## Tech Stack

- Kotlin is the primary programming language for the application.
- Jetpack Compose and Material 3 are used to build the user interface.
- Firebase handles user authentication and cloud data storage through Firestore.
- Retrofit is used to manage network requests and API interactions.
- Modern Android development practices are followed, including MVVM architecture and Clean Architecture.
- Compose Navigation is used to handle transitions between different screens within the app.
